SICK AppSpace Coding Starter Kit

The SICK AppSpace Coding Starter Kit (CSK) provides a starting point for coding SensorApps. It consists of a collection of starter kit modules.
Each module provides a specific functionality, such as TCP/IP, IO-Link, FTP, camera setup, power control, logging, user management and many more.

Modules appear as apps in the SICK AppStudio workspace, provide their key functionalities as served functions/events and optionally contain a user interface (UI).
By using and combining these modules, SensorApp developers do not have to start programming from scratch, but work with a modular, extensible architecture right from the start.

You can find the CSK features, which are currently already installed on this device here.

Visit the SICK AppPool to install additional CSK feature bundles (like the CSK_Communication_Bundle, CSK_ImageProcessing_Bundle, etc.).

Furthermore, some modules even provide the possibility to link their features in a graphical flow configuration to make it possible to solve applications even without any coding.

General information about CSK is available on GitHub .
